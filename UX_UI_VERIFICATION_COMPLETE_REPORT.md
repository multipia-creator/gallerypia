## 🛠️ 기술 상세

### 신규 파일 (4개)

| 파일명 | 크기 | 라인 | 설명 |
|--------|------|------|------|

| `auth-verification.js` | 0 | 0 | 인증 검증 (빈 파일) |

### 신규 백엔드 API (4개)

1. **GET /api/auth/verify**
   - JWT 토큰 검증
   - 사용자 정보 반환
   - 세션 유효성 체크

2. **POST /api/auth/password-reset-request**
   - 비밀번호 재설정 요청
   - 이메일 전송 (개발 모드: 콘솔 출력)
   - 토큰 생성 및 저장

3. **POST /api/auth/password-reset-confirm**
   - 비밀번호 재설정 확인
   - 토큰 검증
   - 비밀번호 해싱 및 업데이트

4. **POST /api/user/profile-image**
   - 프로필 이미지 업로드
   - 파일 검증 (크기/타입)
   - Base64 인코딩 저장

### 신규 프론트엔드 컴포넌트 (11개)

1. **SessionManager** - 통합 세션 관리
2. **DashboardAccessControl** - 대시보드 접근 제어
3. **SocialAuth** - 소셜 로그인
4. **PasswordResetFlow** - 비밀번호 재설정
5. **ProfileImageUpload** - 프로필 이미지 업로드
6. **SearchResultSorting** - 검색 정렬
7. **FilterManager** - 필터 관리
8. **PurchaseFlow** - 구매 플로우
9. **ArtworkComparison** - 작품 비교
10. **OnboardingFlow** - 온보딩
11. **SearchHistory** - 검색 기록
