## 🖼️ NFT 민팅

### **민팅 프로세스 (5단계)**

#### **1단계: 기본 정보**
```
작품명: [필수]
설명: [필수, 최소 20자]
카테고리: [선택]
태그: [선택, 쉼표로 구분]
```

#### **2단계: 작품 업로드**
```
지원 형식: JPG, PNG, GIF, MP4, WEBM
최대 크기: 100MB
권장 해상도: 2048x2048px 이상
```

#### **3단계: 가격 설정**
```
판매 방식:
- 고정 가격
- 경매
- 비공개 (판매 안 함)

가격: ETH 단위
로열티: 0-10% (재판매 수수료)
```

#### **4단계: 메타데이터**
```
속성 추가:
- 키: 값 형식 (예: "Style": "Abstract")
- 최대 10개 속성

IPFS 저장 자동 처리
```

#### **5단계: 민팅 확인**
```
1. 정보 최종 확인
2. MetaMask 서명 요청 승인
3. 가스비 확인 및 승인
4. 민팅 완료 대기 (약 1-2분)
```

### **민팅 비용**
```
가스비: 네트워크 상황에 따라 변동
플랫폼 수수료: 2.5%
IPFS 저장: 무료
```

## ❓ FAQ

### **Q: 가스비는 얼마나 나오나요?**
A: 네트워크 혼잡도에 따라 다르지만, 평균 $5-20 정도입니다. [Gas Tracker](https://etherscan.io/gastracker) 참고.

### **Q: NFT는 어디에 저장되나요?**
A: 작품은 IPFS에, 소유권은 이더리움 블록체인에 저장됩니다.

### **Q: 민팅 후 취소할 수 있나요?**
A: 블록체인에 기록된 후에는 취소 불가능합니다. 신중히 결정하세요.

### **Q: VR 기기가 없어도 되나요?**
A: 네! 일반 브라우저에서도 3D 갤러리를 마우스/키보드로 조작 가능합니다.

### **Q: AR 기능이 작동하지 않아요.**
A: 
1. 카메라 권한 확인
2. 최신 브라우저 사용 (Chrome 권장)
3. HTTPS 연결 확인

### **Q: 수수료는 어떻게 되나요?**
```
판매 수수료: 2.5%
```
