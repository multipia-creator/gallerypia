## 1. 프로그램 개요

### 1.1 프로그램 명칭
**NFT 미술품 5개 모듈 통합 가치산정 시스템**

### 1.2 용도
NFT 디지털 미술품의 객관적이고 과학적인 가치를 평가하기 위한 통합 시스템

### 1.3 개발 환경
- **언어**: TypeScript 5.0+, JavaScript ES2022
- **런타임**: Cloudflare Workers (V8 JavaScript Engine)
- **데이터베이스**: Cloudflare D1 (SQLite)
- **배포 환경**: Cloudflare Pages (Edge Computing)
- **개발 도구**: Vite 5.0+, Wrangler 3.0+

### 1.4 개발 배경
현재 NFT 시장의 주요 플랫폼(OpenSea, Rarible 등)은 단순히 거래 가격만 표시하거나 주관적인 인기도만 제공하여, 미술품의 진정한 예술적 가치를 평가하기 어렵습니다. 본 시스템은 학술 논문을 기반으로 한 과학적 방법론을 통해 이러한 문제를 해결합니다.

## 2. 개발 배경 및 목적

### 2.1 문제 인식
- **문제 1**: NFT 가격의 투기성 및 불투명성
- **문제 2**: 예술적 가치와 시장 가격의 괴리
- **문제 3**: 신진 작가의 정당한 평가 부재
- **문제 4**: 주관적이고 일관성 없는 평가 기준

### 2.2 해결 방안
**5개 모듈 통합 평가 시스템**을 통한 객관적이고 과학적인 가치 산정

### 2.3 목적
1. NFT 미술품의 투명하고 공정한 가치 평가
2. 작가와 구매자 간 신뢰 구축
3. 건전한 NFT 미술 시장 생태계 조성
4. 학술적 방법론 기반 평가 체계 확립

### 2.4 기대 효과
- 투명한 가격 책정 기준 제시
- 신진 작가 발굴 및 육성
- 투자자 의사결정 지원
- NFT 시장 건전성 향상

## 4. 주요 기능 설명

### 4.1 모듈 1: 작가 업적 평가

#### 기능 개요
작가의 경력, 전시 이력, 수상 경력을 정량적으로 평가

#### 평가 항목 (총 15개 변수)
1. **전시 경력** (6개 변수)
   - 개인전 횟수 (가중치: 10점/회)
   - 단체전 횟수 (가중치: 5점/회)
   - 초대전 횟수 (가중치: 8점/회)
   - 국제 전시 횟수 (가중치: 15점/회)
   - 미술관 전시 횟수 (가중치: 12점/회)
   - 갤러리 전시 횟수 (가중치: 7점/회)

2. **수상 경력** (5개 변수)
   - 국제 대회 수상 (권위도 가중치: 3.0)
   - 국가 대회 수상 (권위도 가중치: 2.0)
   - 지역 대회 수상 (권위도 가중치: 1.0)
   - 심사위원장 특별상 (가중치: 2.5)
   - 최다 수상 작가상 (가중치: 1.5)

3. **활동 기간** (2개 변수)
   - 경력 년수 (√년수 × 5점)
   - 최근 활동도 (최근 2년 이내 활동 보너스)

4. **학력 및 소속** (2개 변수)
   - 최종 학력 (박사 30점, 석사 20점, 학사 10점)
   - 소속 기관 권위도 (대학교수 25점, 전문작가 15점)

#### 계산 공식
```
작가업적점수 = (개인전×10 + 단체전×5 + √경력년수×5 + Σ(수상×권위도) + 학력점수 + 소속점수) / 최대점수 × 100
```

#### 최종 점수 범위
0~100점 (정규화)

- 사회적 의미 (0~20점)
- 철학적 깊이 (0~15점)
- 감성적 호소력 (0~15점)
- 스토리텔링 (0~10점)
- 시대적 반영 (0~10점)

**2.2.2 표현성 평가** (7개 변수)
- 기법 완성도 (0~30점)
- 색채 조화 (0~20점)
- 구도 완성도 (0~20점)
- 질감 표현 (0~15점)
- 빛과 그림자 (0~10점)
- 세부 묘사력 (0~10점)
- 시각적 임팩트 (0~15점)

**2.2.3 독창성 평가** (7개 변수)
- 작품 차별성 (0~30점)
- 기법 혁신성 (0~25점)
- 아이디어 참신성 (0~20점)
- 스타일 독자성 (0~15점)
- 실험성 (0~10점)
- 장르 융합도 (0~10점)
- 트렌드 선도성 (0~10점)

**2.2.4 소장가치 평가** (7개 변수)
- 시대적 의미 (0~25점)
- 미래 전망 (0~25점)
- 희소성 (0~20점)
- 작품 상태 (0~15점)
- 보존 가능성 (0~10점)
- 재판매 가능성 (0~10점)
- 투자 가치 (0~15점)

#### 계산 공식
```
작품내용점수 = (내용성점수 + 표현성점수 + 독창성점수 + 소장가치점수) / 4
```

#### 최종 점수 범위
0~100점 (4개 지표 평균)

### 4.3 모듈 3: 저작권/소유권/라이선스 인증 평가

#### 기능 개요
작품의 법적 권리 및 블록체인 인증 상태 평가

#### 평가 항목 (총 12개 변수)

**3.3.1 저작권 인증** (4개 변수)
- 저작권 등록 여부 (25점)
- 등록 번호 유효성 (15점)
- 등록 기관 권위도 (10점)
- 등록 날짜 (최근일수록 높음)

**3.3.2 블록체인 인증** (4개 변수)
- 블록체인 해시 존재 (30점)
- 네트워크 신뢰도 (Ethereum 25점, Polygon 20점 등)
- IPFS 저장 여부 (15점)
- 스마트 컨트랙트 검증 (10점)

**3.3.3 라이선스 평가** (4개 변수)
- 라이선스 유형 (독점 30점, 비독점 15점)
- 라이선스 범위 (전세계 25점, 지역 15점)
- 사용 권한 명확성 (15점)
- 2차 저작물 허용 여부 (10점)

#### 계산 공식
```
인증점수 = (저작권점수 + 블록체인점수 + 라이선스점수) / 최대점수 × 100
```

#### 최종 점수 범위
0~100점 (정규화)

- 현대 기법 숙련도 (0~25점)

- 기술적 완성도 (0~30점)

**4.4.2 미술사적 의미** (4개 변수)
- 미술사 맥락 이해 (0~25점)
- 전통 계승 및 혁신 (0~25점)
- 작가론적 위치 (0~20점)
- 미술사적 기여도 (0~30점)

**4.4.3 시장성** (4개 변수)
- 시장 수요 예측 (0~25점)

- 컬렉터 선호도 (0~20점)

**4.4.4 발전 가능성** (4개 변수)
- 작가 성장 잠재력 (0~25점)
- 작품 시리즈 확장성 (0~25점)
- 브랜드 가치 (0~20점)

#### 전문가 가중치
- 미술관장: 3.0
- 큐레이터: 2.5
- 미술평론가: 2.0
- 박사: 1.5

#### 계산 공식
```
전문가평가점수 = Σ(전문가i점수 × 가중치i) / Σ가중치i
```

#### 최종 점수 범위
0~100점 (가중 평균)

### 4.5 모듈 5: 대중 인지도/인기도 평가

#### 기능 개요
SNS, 미디어, 플랫폼 데이터를 통한 대중적 인기도 정량 평가

#### 평가 항목 (총 12개 변수)

**5.5.1 소셜 미디어 지표** (4개 변수)
- YouTube 조회수 (log10(조회수) × 10점)
- Instagram 참여도 (좋아요 + 댓글 + 저장)
- Twitter 언급량 (리트윗 + 인용 트윗)
- TikTok 영상 조회수

**5.5.2 플랫폼 상호작용** (4개 변수)
- 플랫폼 좋아요 (개수 × 0.5점)
- 플랫폼 댓글 (개수 × 1점)
- 플랫폼 공유 (개수 × 2점)
- 즐겨찾기 추가 (개수 × 1.5점)

**5.5.3 검색 및 미디어** (4개 변수)
- Google Trends 점수 (0~100)
- 언론 보도 건수 (건당 5점)
- 블로그 포스팅 (건당 2점)

```
대중인지도점수 = (SNS점수 + 플랫폼점수 + 미디어점수) / 최대점수 × 100
```

#### 최종 점수 범위
0~100점 (정규화, 로그 스케일 적용)

## 5. 핵심 알고리즘

### 5.1 5개 모듈 통합 가중치 계산

#### 최종 가치 산출 공식
```
최종가치점수 = α1·M1 + α2·M2 + α3·M3 + α4·M4 + α5·M5

여기서:
M1 = 작가 업적 평가 점수 (0~100)
M2 = 작품 내용 평가 점수 (0~100)
M3 = 인증 평가 점수 (0~100)
M4 = 전문가 정성 평가 점수 (0~100)
M5 = 대중 인지도 평가 점수 (0~100)

α1 + α2 + α3 + α4 + α5 = 1 (가중치 합 = 1)
```

#### 기본 가중치 (Default)
```
α1 (작가 업적) = 0.20 (20%)
α2 (작품 내용) = 0.30 (30%) ← 최우선
α3 (인증) = 0.15 (15%)
α4 (전문가 평가) = 0.25 (25%)
α5 (대중 인지도) = 0.10 (10%)
```

#### 가중치 조정 규칙
```
신진 작가: α2 증가 (0.40), α1 감소 (0.10)
유명 작가: α1 증가 (0.30), α5 증가 (0.15)
투자용: α4 증가 (0.35), α2 감소 (0.20)
```

### 5.2 추정 가치 계산

#### 기본 가격 산출
```
기본가격(USD) = 최종가치점수 × 작가등급계수 × 작품크기계수

작가등급계수:
- 신인: 10 USD/점
- 중견: 50 USD/점
- 유명: 200 USD/점
- 거장: 1000 USD/점

작품크기계수:
- 소형 (< 50cm²): 1.0
- 중형 (50~200cm²): 2.0
- 대형 (> 200cm²): 3.5
```

#### 최종 추정 가치
```
추정가치 = 기본가격 × 시장조정계수

시장조정계수 = (1 + 최근거래증가율) × 장르인기도 × 희소성계수
```

#### 최소 권장 가격
```
최소가격 = 추정가치 × 0.7
최대가격 = 추정가치 × 1.3
```

### 7.1 작가 업적 평가 함수

```typescript
/**
 * 모듈 1: 작가 업적 점수 계산
 * @param artist - 작가 정보 객체
 * @returns 0~100 점수
 */
export function calculateArtistAchievement(artist: Partial<Artist>): number {
  const currentYear = new Date().getFullYear();
  
  // 전시 경력 점수
  const soloExhibitionScore = (artist.solo_exhibitions_count || 0) * 10;
  const groupExhibitionScore = (artist.group_exhibitions_count || 0) * 5;
  
  // 수상 경력 점수 (권위도 가중치)
  const authorityWeight = 1.0; // 기본
  const awardsScore = (artist.awards_count || 0) * 15 * authorityWeight;
  
  // 경력 년수 점수 (제곱근 적용으로 초기 년도 가중)
  const careerScore = Math.sqrt(artist.career_years || 1) * 5;
  
  // 최근 활동도 (최근 2년 이내 활동 보너스)
  const latestYear = artist.latest_exhibition_year || 0;
  const recentBonus = (currentYear - latestYear <= 2) ? 10 : 0;
  
  // 합산 및 정규화
  const rawScore = soloExhibitionScore + groupExhibitionScore + 
                   awardsScore + careerScore + recentBonus;
  
  // 0~100 범위로 정규화
  const normalizedScore = Math.min(100, (rawScore / 150) * 100);
  
  return Math.round(normalizedScore * 10) / 10;
}
```

### 7.2 작품 내용 평가 함수

```typescript
/**
 * 모듈 2: 작품 내용 점수 계산 (4가지 지표)
 * @param artwork - 작품 정보 객체
 * @returns 0~100 점수
 */
export function calculateArtworkContent(artwork: Partial<Artwork>): number {
  // 4가지 핵심 지표 (각 0~100점)
  const contentDepth = artwork.content_depth_score || 0;      // 내용성
  const expression = artwork.expression_score || 0;           // 표현성
  const originality = artwork.originality_innovation_score || 0; // 독창성
  const collectionValue = artwork.collection_value_score || 0;   // 소장가치
  
  // 4개 지표 평균
  const averageScore = (contentDepth + expression + originality + collectionValue) / 4;
  
  return Math.round(averageScore * 10) / 10;
}
```

### 7.3 인증 평가 함수

```typescript
/**
 * 모듈 3: 저작권/인증 점수 계산
 * @param artwork - 작품 정보 객체
 * @returns 0~100 점수
 */
export function calculateCertification(artwork: Partial<Artwork>): number {
  let score = 0;
  
  // 블록체인 해시 존재 (30점)
  if (artwork.blockchain_hash) {
    score += 30;
  }
  
  // 저작권 등록 번호 존재 (25점)
  if (artwork.copyright_registration_number) {
    score += 25;
  }
  
  // 라이선스 유형 평가 (25점)
  if (artwork.license_type === 'exclusive') {
    score += 25;
  } else if (artwork.license_type === 'non-exclusive') {
    score += 15;
  }
  
  // IPFS 저장 여부 (20점)
  if (artwork.ipfs_hash) {
    score += 20;
  }
  
  return Math.min(100, score);
}
```

### 7.4 대중 인지도 평가 함수

```typescript
/**
 * 모듈 5: 대중 인지도 점수 계산
 * @param artwork - 작품 정보 객체
 * @returns 0~100 점수
 */
export function calculatePopularity(artwork: Partial<Artwork>): number {
  // 로그 스케일 적용 (큰 숫자 차이 완화)
  const youtubeScore = Math.log10((artwork.youtube_views || 0) + 1) * 8;
  const instagramScore = Math.log10((artwork.instagram_engagement || 0) + 1) * 8;
  const likesScore = Math.log10((artwork.platform_likes || 0) + 1) * 6;
  const sharesScore = Math.log10((artwork.platform_shares || 0) + 1) * 7;
  
  // Google Trends (0~100) 직접 사용
  const trendsScore = (artwork.google_trends_score || 0) * 0.3;
  
  // 언론 보도 (건당 5점)
  const mediaScore = Math.min(30, (artwork.media_coverage_count || 0) * 5);
  
  // 합산
  const totalScore = youtubeScore + instagramScore + likesScore + 
                     sharesScore + trendsScore + mediaScore;
  
  return Math.min(100, totalScore);
}
```

### 7.5 최종 가치 통합 계산 함수

```typescript
/**
 * 5개 모듈 통합 최종 가치 계산
 * @param artwork - 작품 정보 객체 (5개 모듈 점수 포함)
 * @returns 최종 가치 점수 및 추정 금액
 */
export function calculateArtworkValue(artwork: Partial<Artwork>): {
  finalScore: number;
  estimatedValue: number;
  minPrice: number;
  maxPrice: number;
} {
  // 5개 모듈 점수 추출
  const m1 = artwork.artist_achievement_final_score || 0;
  const m2 = artwork.artwork_content_final_score || 0;
  const m3 = artwork.certification_final_score || 0;
  const m4 = artwork.expert_evaluation_final_score || 0;
  const m5 = artwork.popularity_final_score || 0;
  
  // 가중치 (기본값)
  const alpha1 = artwork.weight_artist || 0.20;
  const alpha2 = artwork.weight_artwork || 0.30;
  const alpha3 = artwork.weight_certification || 0.15;
  const alpha4 = artwork.weight_expert || 0.25;
  const alpha5 = artwork.weight_popularity || 0.10;
  
  // 최종 점수 계산
  const finalScore = alpha1 * m1 + alpha2 * m2 + alpha3 * m3 + 
                     alpha4 * m4 + alpha5 * m5;
  
  // 작가 등급 계수 (예: 중견 작가 = 50 USD/점)
  const artistGradeCoefficient = 50;
  
  // 작품 크기 계수
  const width = artwork.size_width || 50;
  const height = artwork.size_height || 50;
  const sizeArea = (width * height) / 10000; // cm² → m²
  const sizeCoefficient = 1 + Math.log10(sizeArea + 1);
  
  // 기본 가격 계산
  const basePrice = finalScore * artistGradeCoefficient * sizeCoefficient;
  
  // 최종 추정 가치 (시장 조정 계수 1.2)
  const estimatedValue = basePrice * 1.2;
  
  // 최소/최대 권장 가격
  const minPrice = estimatedValue * 0.7;
  const maxPrice = estimatedValue * 1.3;
  
  return {
    finalScore: Math.round(finalScore * 10) / 10,
    estimatedValue: Math.round(estimatedValue),
    minPrice: Math.round(minPrice),
    maxPrice: Math.round(maxPrice)
  };
}
```
