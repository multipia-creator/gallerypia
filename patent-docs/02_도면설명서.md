# 도면 설명서

**발명의 명칭**: NFT 미술품의 가치를 평가하는 방법 및 시스템

---

## 도면 목록

- 도 1: 시스템 전체 구성도
- 도 2: 5개 모듈 통합 평가 프로세스 순서도
- 도 3: 모듈 1 (작가 업적 평가) 상세 프로세스
- 도 4: 모듈 2 (작품 내용 평가) 상세 프로세스
- 도 5: 모듈 3 (인증 평가) 상세 프로세스
- 도 6: 모듈 4 (전문가 평가) 상세 프로세스
- 도 7: 모듈 5 (대중 인지도 평가) 상세 프로세스
- 도 8: 최종 가치 통합 계산 프로세스
- 도 9: 작가 유형별 가중치 조정 프로세스
- 도 10: 평가 결과 출력 화면

- (120): 데이터베이스 - Cloudflare D1 기반 SQLite
- (130): 5개 평가 모듈

  - (132): 모듈 2 - 작품 내용 평가
  - (133): 모듈 3 - 인증 평가
  - (134): 모듈 4 - 전문가 평가
  - (135): 모듈 5 - 대중 인지도 평가
- (140): 통합 계산 모듈 - 가중치 적용 및 최종 가치 산출
- (150): 결과 출력부 - 평가 리포트 생성
