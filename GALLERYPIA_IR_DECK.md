## 1. Executive Summary

### 🎯 회사 개요

**GALLERYPIA (갤러리피아)**는 AI 기반 가격 예측과 전문가 평가 시스템을 결합한 **차세대 NFT 미술품 거래 플랫폼**입니다.

### 💡 핵심 가치 제안

- **AI 가격 예측**: 머신러닝 기반 작품 가치 자동 산정
- **전문가 크라우드소싱**: ETH 보상 기반 분산형 평가 시스템
- **AR/VR 전시**: 메타버스 갤러리 경험
- **NFT 아카데미**: 아티스트 교육 및 커뮤니티

### 📊 핵심 지표 (프로젝트 상태)

| 항목 | 현황 |
|------|------|
| **개발 현황** | 프로토타입 완료 (100%) |
| **기술 스택** | Cloudflare Workers, Hono, TypeScript |
| **코드 규모** | 28,000+ 라인 |
| **지원 언어** | 4개 언어 (한·영·중·일) |
| **배포 상태** | Cloudflare Pages 배포 완료 |
| **도메인** | gallerypia.com 확보 |

### 🎯 투자 요청

- **투자 규모**: 30억원 (Series A)
